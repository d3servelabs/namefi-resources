**ERC-721** est la norme [Ethereum](/fr/glossary/ethereum/) pour les jetons non fongibles ([NFT (Jeton Non Fongible)](/fr/glossary/nft/)). Elle définit une interface commune — propriété, transfert, approbation — sur laquelle les portefeuilles, les places de marché et d'autres contrats peuvent s'appuyer. Parce que les [domaines tokenisés](/fr/blog/what-are-tokenized-domains/) sont généralement implémentés sous forme de NFT ERC-721, ils bénéficient d'une compatibilité immédiate avec l'écosystème NFT au sens large : OpenSea, Blur et des places de marché similaires peuvent les référencer ; les portefeuilles standards comme MetaMask, Rabby et les portefeuilles matériels peuvent les détenir ; les protocoles de prêt et de location compatibles NFT peuvent s'y connecter. Il existe également une norme de lot connexe, ERC-1155, utilisée pour les jetons fongibles ou semi-fongibles — les domaines ne sont généralement pas émis sous ERC-1155 car chaque domaine est unique. Référence : [spécification EIP-721](https://eips.ethereum.org/EIPS/eip-721).
