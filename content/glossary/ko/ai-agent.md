**AI 에이전트**란 대형 언어 모델(또는 유사한 AI)을 기반으로 하여 사용자를 대신해 실제 환경에서 다양한 작업을 수행하는 소프트웨어입니다. 항공편 예약, 구매, 메시지 전송, API 호출은 물론, 점차 다른 에이전트와의 거래까지 수행할 수 있습니다. 텍스트만 생성하는 챗봇과 달리, 에이전트는 상태를 읽고, 추론하며, 결과를 다시 기록합니다. 에이전트가 데모 단계를 넘어 실질적인 경제 활동으로 확장됨에 따라, 지속적인 식별자와 결제 수단이 필요하게 되었습니다. 이 분야의 표준화 노력으로는 Anthropic의 [Model Context Protocol](https://modelcontextprotocol.io)과 새롭게 부상하는 에이전트 커머스 프로토콜이 있습니다. 에이전트 신원 패턴이 어떻게 구축되고 있는지는 [2026년 토큰화 도메인 활용 사례](/ko/blog/tokenized-domain-use-cases-2026/)를 참고하시기 바랍니다.
