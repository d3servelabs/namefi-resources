如果您在 2026 年正在寻找域名代币化平台，您可能会关注到以下几个名字：[Namefi](https://namefi.io)、[Doma Protocol](https://doma.xyz)、D3 Global Inc（也写作 [D3.inc](https://d3.inc) 或 D3 Inc）、3DNS、[Domora](https://domora.com)、[WebUnited](https://webunited.com) 以及 [GBM](https://gbm.auction)。它们的主页上都写着“[代币化域名](/zh/glossary/tokenized-domain/)”（Tokenized domains），但它们的功能其实不尽相同。

## 快速概览

大致上，这些平台可以分为三大阵营：

1. **面向所有者的代币化服务。** 您自带一个真实的 ICANN 域名（或直接注册一个），平台将其代币化。示例：**Namefi，3DNS**。
2. **协议层与面向注册局的基础设施。** 平台构建标准、[智能合约](/zh/glossary/smart-contract/)或注册局合作伙伴关系，供其他平台（及注册商）在此基础上进行开发。示例：**Doma Protocol，D3 Global Inc**。
3. **专门的销售与流动性工具。** 专注于拍卖、碎片化和借贷——这类工具建立在代币化域名之上，而不是去生成代币化域名。示例：**GBM**（拍卖）、**Doma Prime**（流动性）、**Domora**（专注于碎片化）。

这些类别会有所重叠，有些平台甚至跨越了不止一个领域。但“这个平台属于哪个阵营？”永远是您需要考虑的第一个问题。

## 各平台的强项

### Namefi

**最适合：** 想要将真实的 `.com`/`.xyz`/`.io` 域名在以太坊（Ethereum）或 Base 链上[代币化（tokenize）](/zh/glossary/tokenize/)的所有者；不仅能获得广泛的 NFT 市场与 [DeFi](/zh/glossary/defi/) 借贷支持，其 DNS 管理体验也绝不逊色于 Cloudflare。

**主要特点：** 支持众多 [TLD（顶级域）](/zh/glossary/tld/) 且受 [ICANN](/zh/glossary/icann/) 认可的域名；通过标准 [NFT](/zh/glossary/nft/)（[ERC-721](/zh/glossary/erc-721/)）实现链上所有权，完美兼容各种钱包、市场和链上工具；提供包含 [DNSSEC](/zh/glossary/dnssec/) 的全面 DNS 管理功能；内置去中心化应用[市场（marketplace）](/zh/glossary/marketplace/)；并集成了链上支付（[x402](/zh/glossary/x402/)）。支持多链，从第一天起即可实现自托管。

**较不适合：** 想要一个自己尚未拥有的全新 TLD 的用户，或者只想要像 `name.eth` 这样的 [Web3](/zh/glossary/web3/) 原生名称的用户。

### Doma Protocol

**最适合：** 开发者与协议层面的工作。Doma Protocol 是一个协议层——它为代币化 DNS 域名提供共享标准和原语，包括流动性原语（Doma Prime）以及新代币化域名的启动平台（Mizu）。

**主要特点：** 非常侧重于开发者；与多个注册商建立了合作伙伴关系；在其协议之上构建的应用生态系统正在不断增长。

**较不适合：** 只是想将手中现有域名代币化、且不打算搭建协议栈的普通所有者。通常，您会通过构建在 Doma Protocol 之上的平台来间接使用它。

### D3 Global Inc

**最适合：** 注册局层面的合作，以及专为 Web3 用例设计的全新 TLD。

**主要特点：** D3 Global Inc 的定位围绕新的 TLD 和注册局协议，强调 Web2 DNS 与 Web3 命名空间之间的互操作性。

**较不适合：** 如果您的目标是“将我已经拥有的 `.com` 代币化”，D3 Global Inc 并不是最直接的途径——他们更关注注册局/TLD 层面，而非个人所有者的工作流。

### 3DNS

**最适合：** 想要简化代币化流程且不需要涵盖所有冷门 TLD 的 Web3 原生所有者。

**主要特点：** 受 ICANN 认可的域名代币化，UX（用户体验）干净简洁且有明确的导向性。拥有活跃的生态系统合作伙伴。

**较不适合：** 需要广泛 TLD 支持或 Namefi/Doma 所涵盖的某些特定集成的所有者。在做出决定前，值得先对比一下各自的 TLD 支持情况。

### Domora

**最适合：** 碎片化用例——即优质域名的共同所有权。

**主要特点：** 明确以“将域名作为碎片化资产，而不仅仅是可交易的 NFT”为核心理念驱动。

**较不适合：** 追求直接了当“我想代币化我的域名并使用它”的普通工作流。Domora 更多的是关于特定的市场结构，而不是一个通用平台。

### WebUnited

**最适合：** 主要对 Web2 和 Web3 之间的 DNS 镜像桥接感兴趣的所有者。

**主要特点：** 传统 DNS 与链上命名系统之间的镜像式集成模型。

**较不适合：** 想要完整的市场交易平台、借贷技术栈或广泛的 NFT 市场兼容性的用户。

### GBM (GBM Auctions)

**最适合：** 通过“竞价赚取（bid to earn）”拍卖机制销售代币化域名。

**主要特点：** 提供的是拍卖基础设施，而非代币化平台本身。通常与上述某个平台结合使用来处理销售环节。

**较不适合：** 销售环节之外的任何其他用途。

## 对比清单（实用指南）

当您在这些平台之间做决定时，光看营销页面是找不到答案的。以下这些问题能帮到您：

- **TLD 覆盖范围。** 平台是否支持您关心的特定 TLD（如 `.com`、`.io`、`.xyz`、`.art`，或类似 `.de`/`.uk` 的 [ccTLD](/zh/glossary/cctld/)）？
- **区块链覆盖范围。** 是以太坊主网、Base、Polygon 还是其他链？NFT 存储在哪里？Gas 费用在哪里支付？
- **NFT 标准。** 是否使用标准的 ERC-721？这关系到市场的兼容性（如 OpenSea、Blur、Magic Eden 等）以及链上借贷是否可用。
- **DNS 管理。** 您能否在平台内管理 DNS？是否可以使用外部[域名服务器](/zh/glossary/nameserver/)（如 Cloudflare、Route53）？是否支持 DNSSEC？
- **借贷 / 抵押。** 您是否可以在现有的货币市场上将代币化域名作为抵押品借款，还是只能在平台特定的孤岛体系内进行？
- **市场兼容性。** Blur、OpenSea 等平台会显示您的挂单吗，还是只能在该平台自带的市场内看到？
- **托管模式。** 是自托管（掌握在您自己的钱包，用您的私钥，自行负责）还是平台托管？两者各有优劣。
- **续费流程。** 谁向注册商付款？年度续费如何计费？如果您停止付款会发生什么？
- **退出路径。** 如果某天您想要*解除代币化（de-tokenize）*并回到普通的注册商设置，是否可以实现？
- **费用。** 铸造费、市场交易费、转移费、续费以及 Gas 费。在做出决定前，请根据您的具体情况将它们加起来算一算。

如果一个平台的官方文档没有说清楚上述内容中的任何一项，这本身就是一个值得注意的信号。

## 诚实的权衡矩阵

| 如果您想要… | 请重点了解 |
|---|---|
| 将您已拥有的真实 `.com`/`.xyz`/`.io` 代币化，获得广泛的市场和借贷支持，以及完善的 DNS 管理 | **Namefi** |
| 作为开发者，在代币化域名协议*之上*进行构建 | Doma Protocol |
| 全新的 TLD 和注册局级别的合作伙伴关系 | D3 Global Inc |
| 简化的代币化用户体验与明确的流程设计 | 3DNS, Namefi |
| 优质名称的碎片化 / 共同所有权 | Domora |

| 拍卖式的销售基础设施 | GBM |

## 关于“最佳”平台

没有绝对通用的“最佳”平台。正确的答案取决于：

- 您关心哪些 TLD。
- 您是域名所有者、开发者还是市场平台方。
- 您有多看重资产的自托管权。
- 您是想将域名用作 DeFi 抵押品，还是仅仅想持有它。
- 您对任何单一平台的信任程度。

我们显然认为，对于许多域名所有者来说，Namefi 是正确的选择。但您能做的最好的事情是，**除了我们之外，至少再尝试使用一家其他平台**。如果我们更好，您在横向对比之后自然会得出结论。如果其他平台更适合您的使用场景，您就应该果断选择它。

## 总结

- 域名代币化平台分为三大类：面向所有者的服务、协议层以及专门的销售/流动性工具。
- **Namefi** 和 **3DNS** 对于想要将手中现有域名代币化的所有者来说，是最直接的途径。
- **Doma Protocol** 是协议层；通常情况下您会间接地与其进行交互。
- **D3 Global Inc** 专注于注册局层面，主要面向新的 TLD。
- **Domora**、**WebUnited** 和 **GBM** 分别在碎片化、DNS 桥接和拍卖方面具备专长。
- 正确的平台选择取决于您所需的 TLD、对托管方式的偏好、市场的兼容性要求以及您打算如何*使用*这些代币化域名。
- 在将您的资产组合托付给某个平台之前，请多尝试几家。
