在 Box 成为一家价值数十亿美元的企业软件公司之前，它的官网地址稍微欠缺一点权威性：**Box.net**。

这个名字是一种妥协，而非首选。当 [Aaron Levie（南加州大学商科学生）在 2003 年萌生创立 Box.com 的想法](https://en.wikipedia.org/wiki/Box_(company)#:~:text=The%20idea%20for%20Box.com%20started%20in%202003%20with%20Aaron%20Levie) 时，那个干净利落且完全匹配的域名——Box.com——已经被注册了。因此，这家后来成为 [Box, Inc.（前身为 Box.net）](https://en.wikipedia.org/wiki/Box_(company)#:~:text=Box%2C%20Inc.%20(formerly%20Box.net)) 的公司，做了无数初创公司在 .com 域名被抢注时都会做的事：退而求其次，选择第二好的顶级域名（TLD），以 Box.net 上线并发布了产品。

为什么选 "Box.net"？因为 Box.com 属于别人——具体来说，属于由一位前西瓜农场主经营的优质域名控股公司 Digimedia。大家*理所当然*认为该公司应该拥有的那个完全匹配的域名，其实正躺在一个域名投资人的投资组合里，遥不可及，直到 Box 有了足够的资金和理由去拿下它。

在其发展初期，.net 域名完全够用。[2005 年，Levie 辍学，与相识多年的好友兼联合创始人 Dylan Smith 全职投入 Box 的工作](https://en.wikipedia.org/wiki/Box_(company)#:~:text=In%202005%2C%20Levie%20dropped%20out%20of%20school%20to%20work%20on%20Box%20full-time%20with%20long-time%20friend%20and%20cofounder%20Dylan%20Smith)，随后 [Levie 和 Smith 又迎来了联合创始人 Jeff Queisser 和 Sam Ghods](https://en.wikipedia.org/wiki/Box_(company)#:~:text=Levie%20and%20Smith%20were%20joined%20by%20cofounders%20Jeff%20Queisser%20and%20Sam%20Ghods)。当时的商业模式是简单的消费者存储服务：正如 Levie 后来所描述的那样，[“人们每月付给我们 2.99 美元，我们给他们 1GB 的存储空间”](https://nira.com/box-history/#:~:text=People%20would%20pay%20us%20%242.99%20a%20month%2C%20we%E2%80%99d%20give%20them%201GB%20of%20storage)。这个具有描述性且可用的域名，承载着早期的产品，一切都刚刚好。

后来，在 2011 年，Box 终于买下了 **Box.com**。正如 Domain Name Wire 报道的那样，[文件共享服务商 Box.net 从 Digimedia 的 Scott Day 手中买下了 Box.com 域名](https://domainnamewire.com/2012/03/25/really-cool-guys-at-digimedia-sold-box-com-for-about-1-million/#:~:text=file%20sharing%20service%20Box.net%20bought%20the%20Box.com%20domain%20name%20from%20Scott%20Day%20at%20Digimedia)——并且[该公司为购买该域名支付了 90 万至 99.9 万美元](https://domainnamewire.com/2012/03/25/really-cool-guys-at-digimedia-sold-box-com-for-about-1-million/#:~:text=the%20company%20bought%20the%20domain%20for%20between%20%24900%2C000%20and%20%24999%2C000)。

这是一个初创公司的故事：它以 .net 作为折中方案上线，在最初的六年里逐渐成长为一个真正的品牌，并最终支付了近百万美元去掉了 ".net"——而这正好发生在它不再叫 "Box.net" 并简单地更名为 **Box** 的关键节点。

## 2005年：推动产品上线的 .net 折中方案

一开始，".net" 是一种限制，而不是一种策略。

产品理念来源于 Levie 亲眼目睹的一个现实问题。根据公开的创业史，[2004 年，尚未确定大学专业的 Levie 接受了洛杉矶派拉蒙影业（Paramount Pictures）的实习。正是在派拉蒙工作期间，Levie 亲身体会到了制片厂共享大文件有多么困难](https://nira.com/box-history/#:~:text=In%202004%2C%20with%20his%20major%20still%20undecided%2C%20Levie%20accepted%20an%20internship%20at%20Paramount%20Pictures)。解决办法就是在线文件存储和共享——一个存在于互联网上的“盒子（box）”。这个名字堪称完美。只可惜对应的 .com 域名无法获得。

于是 Box 在 Box.net 上线了。对于那些每月花几美元购买 1GB 空间的早期尝鲜者来说，顶级域名（TLD）几乎无关紧要：

- 品牌词仍然是 "Box"——简短、易记且具有独占性。
- 产品与名称相符：一个在线存放文件的盒子。
- 解释成本很低，这对于一家既没有品牌资产又几乎没有资金的公司来说至关重要。

事实上，资金正是创立之初的最大制约因素。为了让服务上线，[Levie 和 Smith 动用了 Smith 存下来的 15,000 美元在线扑克赢利来租用服务器空间](https://nira.com/box-history/#:~:text=Levie%20and%20Smith%20used%20%2415%2C000%20in%20online%20poker%20winnings%20that%20Smith%20had%20been%20saving%20to%20rent%20server%20space)。第一笔真正的支票来自一位如今已名声大噪的早期支持者：[创始人最初依靠自有资金以及亲朋好友的支持，直到 2005 年 Mark Cuban 投资了 35 万美元的种子资金](https://en.wikipedia.org/wiki/Box_(company)#:~:text=Mark%20Cuban%20invested%20%24350%2C000%20in%20seed%20funding%20in%202005)。一家靠打扑克赢来的钱和一张单一天使支票维持运转的公司，绝不可能把它的发展资金用来从域名投资人的投资组合中撬出一个高价的优质 .com 域名。Box.net 是他们当时唯一能负担得起的地址。

更关键的是，该公司无论如何也拿不到那个干净利落的版本。Box.com 并非偶然闲置——它是互联网上最著名的优质域名投资组合之一中被精心培育的资产。Box.net 不是品牌上的点缀；它是当时可用的最佳地址，而 Box.com 标着高价，年轻的初创公司在当时还无法证明这笔开销的合理性。

## 2011年：买下梦寐以求的域名

到了 2011 年，Box 遇到了一个只有成功才会带来的烦恼：那个*更好的*域名地址就摆在那里，而公司的发展已经超越了 .net 的承载能力。

Box 已经完成了从廉价消费者存储向企业级服务的重大转型。在域名升级时，TechCrunch 将 Box 描述为[一个带有协作、社交和移动功能的企业云存储平台](https://techcrunch.com/2011/09/28/box-net/#:~:text=cloud%20storage%20platform%20for%20the%20enterprise%20that%20comes%20with%20collaboration%2C%20social%20and%20mobile%20functionality)，并报道称[拥有 700 万用户、存储 3 亿份文档的 Box](https://techcrunch.com/2011/09/28/box-net/#:~:text=Box%2C%20which%20has%207%20million%20users%20and%20stores%20300%20million%20documents)，已经被 [10 万家企业采用，其中包括 77% 的财富 500 强企业](https://techcrunch.com/2011/09/28/box-net/#:~:text=100%2C000%20businesses%2C%20including%2077%25%20of%20the%20Fortune%20500)。一家面向绝大多数财富 500 强企业销售产品的公司，不再适合使用一个听起来像业余文件托管网站的域名了。

于是它买下了完全匹配的域名。卖家是一位专业人士：[西瓜农场主 Scott Day 创立了 Digimedia。他通过购买 Watermelons.com 进入了域名行业](https://domainnamewire.com/2012/03/25/really-cool-guys-at-digimedia-sold-box-com-for-about-1-million/#:~:text=Watermelon%20farmer%20Scott%20Day%20founded%20Digimedia)。多年后，一份域名指南用一句话总结了这笔交易：[2011 年，互联网公司 Box 的所有者支付了近 100 万美元，完成了从 Box.net 到 Box.com 的升级](https://jamesnames.com/guide/digimedia/#:~:text=In%202011%2C%20the%20owners%20of%20the%20Internet%20company%20Box%20paid%20close%20to%20%241%20million%20to%20upgrade%20from%20Box.net%20to%20Box.com)。

花费近百万美元买下的并不是一个虚荣的网址。这是为了最终拥有与用户下意识输入的品牌完全匹配的官网地址而付出的代价。

## “非常酷的家伙”和不急于出手的卖家：谈判内幕

![Vivid full-color editorial illustration of a Box-blue .net box transforming into a glowing Box.com box, money changing hands between a premium-domain seller and the Box team, in Box blue 0061D5](../../assets/from-box-net-to-box-com-02-acquisition.jpg)

买下 Box.com 之所以花了六年时间，原因和大多数优质域名交易进展缓慢的原因一样：所有者并不急于出售。

Digimedia 并不是靠注册错别字域名来囤积居奇的投机者。它是一个专业的投资组合，持有并出售了互联网上一些最通用、最有价值的单字域名。出售 Box.com 是一场按照卖家意愿进行的交易，而不是急于脱手的套现。定价也反映了这一点——对于一个作为 .com 的常见英语单词，售价在 90 万到 99.9 万美元之间。

值得注意的是，考虑到卖方占据着主导权，这笔交易的过程可以说是非常*融洽*的。CNET 作家 [Rafe Needleman 与 Box 首席执行官 Aaron Levie 讨论了这个域名](https://domainnamewire.com/2012/03/25/really-cool-guys-at-digimedia-sold-box-com-for-about-1-million/#:~:text=CNET%20writer%20Rafe%20Needleman%20discussed%20the%20domain%20name%20with%20Box%20CEO%20Aaron%20Levie)，结论几乎是充满温情的：[谈及与 Digimedia 的谈判时，Levie 表示 Digimedia 的人‘非常酷’](https://domainnamewire.com/2012/03/25/really-cool-guys-at-digimedia-sold-box-com-for-about-1-million/#:~:text=Levie%20said%20the%20Digimedia%20guys%20were)。这里没有香槟庆功宴与法庭诉讼交织的抓马剧情——只有一个明确知道自己想要什么的资金充裕的买家，和一个明确知道自己持有何种价值资产的专业卖家。

而 Box 非常渴望得到它，甚至愿意打破预算。根据同一份报道，[如果有必要，Levie 准备为这个域名花更多的钱](https://domainnamewire.com/2012/03/25/really-cool-guys-at-digimedia-sold-box-com-for-about-1-million/#:~:text=Levie%20was%20prepared%20to%20spend%20more%20for%20the%20domain%20if%20he%20had%20to)。这表明这绝不是一次粉饰门面的购买。一位已经筹集了巨额企业级融资的创始人，愿意支付*高于*近七位数报价的价格，说明他已经认定这个完全匹配的 .com 域名是基础设施，而不是装饰品。

这个故事与典型的域名抢注故事的区别在于没有戏剧性的冲突，而这正是关键所在。当卖方拥有合法的投资组合，买方是有实力的正规公司时，“谈判”的主要内容就是为一个缺乏公开可比价格的资产寻找合理定价。摩擦的根源不是敌意，而是估值。

## 当时的这笔巨款有着不同的意义

事后看来，近 100 万美元的价格可以说是相当划算了。Box 后来在纽约证券交易所上市，成为一家价值数十亿美元的企业软件公司；Box.com 如今是其最默默无闻却最不可或缺的永久资产之一。相比之下，一百万美元就像是财务报表上的舍入误差。

但我们应该将其放在花费这笔钱的那个特定时刻来评估。

2011 年，Box 是一家快速发展但仍未上市的公司，正处于争夺企业市场的昂贵圈地运动中。就在六年前，这家公司还是靠着打扑克赢来的钱和 35 万美元的天使投资起步的。在达成这笔域名交易前后，[Box 刚刚又筹集了 5,000 万美元的发展资金](https://techcrunch.com/2011/09/28/box-net/#:~:text=Box%20just%20raised%20another%20%2450%20million%20in%20growth%20capital)——这些钱是用来资助销售团队、基础设施建设和开发企业级功能的，而不是用来买网址的。

在这种背景下，花费近 100 万美元购买一个*域名*——而不是聘请工程师、建设数据中心或雇佣企业销售代表——是一项重大的资本配置决策。只有当你将完全匹配的 .com 域名视为基础设施时，这一切才说得通：每一个首席信息官（CIO）、每一次媒体提及、每一个集成合作伙伴，以及每一份财富 500 强企业的采购表单，都会访问这个地址。Box 押注的是，一家要求大型企业信任并托付其文件的公司，绝不应该挂靠在一个 .net 域名下。

## 为什么从 .net 升级到 .com 至关重要

![Vivid full-color editorial illustration of a Box-blue cardboard box labeled .net peeling off its suffix and becoming a brighter, more solid box labeled .com, in Box blue 0061D5 on a clean white background](../../assets/from-box-net-to-box-com-01-net-to-com.jpg)

Box.net 和 Box.com 之间的差距只是三个字母。但从战略上讲，这是“碰巧可用的版本”与“大家都认为是正牌的版本”之间的本质区别。

**Box.net** 传递出一种退而求其次的信号——这是当 .com 被占用时你被迫选择的地址。**Box.com** 则代表着默认值、正统地位，是用户不假思索就会直接输入的地址。对于一个面向消费者的玩具级产品来说，这种差距是可以忍受的。但对于一家向财富 500 强企业推销信任感和持久性的公司来说，它就是一种隐形且持续的阻碍。

| 之前 | 之后 |
| --- | --- |
| Box.net | Box.com |
| 听起来像是备用顶级域名 | 听起来像是最正统的官网地址 |
| 当 .com 被抢注时无奈选择的版本 | 大家都下意识认为的官方版本 |
| 带来微妙的疑虑：“他们是官方正品吗？” | 天然自带企业买家的默认信任感 |
| 容易被错拼成 Box.com——然后跳出页面 | 就是那个用户本能猜出的网址 |

这是所有域名升级案例中都会出现的共同规律：早期的地址代表着*妥协*，而极佳的地址意味着*掌控*。当完全匹配的 .com 域名被占用或由于价格原因无法获得时，选择一个不同的 TLD（如 .net、.io、.co）或添加描述性修饰词，是完全合理的过渡方案。一旦品牌变得足够强大，并且公司资金足够雄厚，拿下代表自身名字正统版本的域名，这项升级自然会获得丰厚的回报。

对于 Box 来说，.net 从来都不是他们的终极目标。那只是该公司在 2005 年唯一能负担得起的地址，之所以一直保留，仅仅是因为 Box.com 挂着一个靠扑克奖金起家的初创公司无法承担的高价。

## 域名终于追上了公司的脚步

时机说明了一切。域名升级与企业品牌形象升级是同步进行的。

在最初的六年里，Box 的产品、品牌和野心都在彰显 "Box"，而它的官网地址却是 "Box.net"——随着签下的企业客户越来越多，这种错位感也变得愈发尴尬。2011 年的升级解决了这个问题：公司拿下了 Box.com，并在同一时期正式更名为 [Box, Inc.（前身为 Box.net）](https://en.wikipedia.org/wiki/Box_(company)#:~:text=Box%2C%20Inc.%20(formerly%20Box.net))——正如域名升级让一切成为可能一样，品牌形象也去掉了 ".net"。

想象一下另一种情况：一家公司试图说服 77% 的财富 500 强企业将重要文档托付给它，而它最正统的网络地址仍然是 Box.net，Box.com 却指向别处。除非买下 Box.com，否则品牌无法完全聚焦在 "Box" 上。这个进展缓慢、被外部持有且价格昂贵的拼图——域名——必须先被锁定，那个简洁、单一词汇的品牌形象才能最终确立。

这就是为什么这次升级不是在做表面文章。就在那一刻，产品名称、公司名称和网站地址终于统一成了相同的三个字母，并且落在了每个人都期望看到的顶级域名上。

## 域名成为了操作系统的一部分

优质域名无关乎虚荣。它们关乎重复率——而且对于企业供应商来说，关乎信任。

公司的核心域名会出现在营销团队无法直接控制的地方：

- 发送给客户的每一位员工电子邮件地址和签名中。
- 在安全审查、采购表格和供应商入驻流程中。
- 在新闻标题、分析师报告和系统集成目录中。
- 在搜索结果和浏览器地址栏中。
- 在管理员们口口相传的每一次推荐中。

这里的每一次重复，要么增加摩擦，要么减少阻力。Box.net 让每一次提及都带有一丝淡淡的疑虑——*.net 是官方正版吗，还是说我应该使用 Box.com？*——而且很容易让人错输成 Box.com，从而跳转到一个完全不属于该公司的停放页面。而 Box.com 使每一次提及都变得权威且具有自我纠错能力：人们习惯性猜测的地址，现在直接指向了真正的公司。

将这种摩擦放大到 700 万用户、10 万家企业和大多数财富 500 强企业身上——对于这些买家来说，“这家供应商连自己的 .com 域名都没有吗？”是一个真实存在但可能不宣之于口的信任危险信号——这时，近 100 万美元看起来就不再是一件奢侈品了。它是一次永久性的阻力消除，以及填补了企业级信誉度上的空白。

## 创始人应该从这第 17 个案例中学到什么

最直接的结论——“在第一天就买下完全匹配的 .com 域名”——其实是错误的。Box *根本办不到*；那个 .com 是一个被精心持有的优质资产，而这家公司最初的资金是靠赢来的扑克奖金。更有用的经验在于把握阶段性策略：

1. **用不同的顶级域名（TLD）上线完全没问题。** Box.net 承载了一个消费者产品、一次向企业级方向的转型、数百万用户以及 5,000 万美元的发展资金。使用 .net、.io 或 .co——或者添加一个描述性修饰词——并不是失败。当 .com 被占用或无法负担时，这是合理的过渡方案。
2. **密切关注当前 TLD 开始让你付出代价的那个临界点。** Box 释放信号并非出于美观考虑。当他们用一个 .net 域名向财富 500 强推销服务时——“我们现有的地址”与“买家信赖的地址”之间的差距，便成了一种信誉税。
3. **把完全匹配的 .com 域名当成基础设施，并为其预留预算。** 决定升级其实并不难。创始人是用发展资金支付了这近 100 万美元的实际成本，并且[准备在必要时花更多钱](https://domainnamewire.com/2012/03/25/really-cool-guys-at-digimedia-sold-box-com-for-about-1-million/#:~:text=Levie%20was%20prepared%20to%20spend%20more%20for%20the%20domain%20if%20he%20had%20to)。
4. **从专业卖家那里购买，要做好接受专业报价的心理准备。** 这里没有需要诉诸法律解决的域名抢注纠纷——只有 Digimedia，一个合法持有单一词汇 .com 的投资组合。谈判过程非常[融洽](https://domainnamewire.com/2012/03/25/really-cool-guys-at-digimedia-sold-box-com-for-about-1-million/#:~:text=Levie%20said%20the%20Digimedia%20guys%20were)；唯一的分歧点在于价格。

域名升级并没有让 Box 赢得比赛。产品、企业转型的战略、时机和执行力要重要得多。但是，Box.com 让公司的名称、产品和网址最终达成了统一——并且消除了每一次客户互动中潜藏的疑问：“为什么一家企业级供应商在用 .net 域名？”

## Namefi 视角

![Colorful illustration of a premium domain moving through verified transfer, a green Namefi token, and DNS continuity](../../assets/from-box-net-to-box-com-03-namefi-angle.jpg)

从核心上看，Box 的故事是一个资产转移与估值的问题。

战略决策方面从没有任何疑问——一家叫 Box 的公司当然应该拥有 Box.com。难点在于围绕这项资产的方方面面：确认这个 .com 归属于一个优质域名投资组合，为一个没有公开可比价格的单字域名谈妥价格，[准备支付甚至高于](https://domainnamewire.com/2012/03/25/really-cool-guys-at-digimedia-sold-box-com-for-about-1-million/#:~:text=Levie%20was%20prepared%20to%20spend%20more%20for%20the%20domain%20if%20he%20had%20to)近七位数的报价，并在不中断 10 万家企业正在使用的在线服务的情况下，干净利落地完成控制权的转移。六年时间之所以漫长，并不是因为*做决定*难，而是因为*执行*难——即为一项优质资产进行定价和转移。

[Namefi](https://namefi.io) 的创立理念正是基于域名应该像互联网原生资产一样运作。代币化的所有权可以使域名的控制更容易验证、转移并集成到现代工作流中，同时保持与 DNS 的兼容性——这将类似交易中最混乱的部分（证明谁真正控制了一个优质域名、就价值达成共识以及安全转移资产）转化为一种更接近于干净、可审计的交易过程。

Box.com 现在的地位看起来是必然的，因为 Box 已经长成了庞然大物。但真正的经验早在公司达到那个规模之前就已显现：当一个名字即将承载整个业务——尤其是一家要求大型企业给予信任的业务时，完全匹配的 .com 域名绝对不是装饰品。它是品牌价值近百万美元的核心资产，值得你不惜一切代价，最终把它收入囊中。
