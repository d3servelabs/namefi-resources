## **Namefi エコシステムに BlockEden.xyz が参加**

BlockEden.xyz は、従来型ドメインをオンチェーンへ橋渡しするという Namefi のミッションを支援しています。Namefi が Web2 DNS 資産を NFT としてトークン化し、互換性と使いやすさを保ちながらドメイン所有を再定義するなかで、BlockEden.xyz はそうした革新を安心して拡張するためのインフラ基盤を提供します。

### **BlockEden.xyz とは？**

BlockEden.xyz は、Web3 の拡張性と信頼性を重視した開発者向けインフラプラットフォームです。サンフランシスコに本社を置く BlockEden.xyz は、Aptos、Sui、Solana、十数種類の EVM 互換ブロックチェーン上で開発する開発者に、高可用性 RPC ノード、インデクサー、分析 API を提供しています。

99.9% の稼働率、エンタープライズ級の API スループット、5,000 万ドル超のステーク済み資産により、BlockEden.xyz は世界中の数千の開発者とプロジェクトから信頼されています。

中核となるインフラに加えて、BlockEden は暗号資産決済ソリューションの [Eden Pay](https://blockeden.xyz/docs/eden-pay/) と包括的な [API Marketplace](https://blockeden.xyz/api-marketplace/) を提供しています。また、[Blockroma explorer](https://blockroma.com/) のようなオープンソースの取り組みも支援し、Web3 エコシステムを構築するための総合的なツールキットを開発者に届けています。

### **Namefi にとって重要な理由**

Namefi は `example.com` のようなドメインを ERC-721 NFT としてミントできるようにし、DNS の使いやすさを保ちながら新しいオンチェーン機能を解放します。この進化には、以下を支える堅牢なインフラが必要です。

* トークン化ドメインとの大量の読み書きインタラクションを処理する

* NFT マーケットプレイスとドメイン取引プロトコルを動かす

* ドメイン所有履歴のインデックス作成、分析、イベント追跡を支援する

* ドメインを担保として使うような将来の DeFi ユースケースを可能にする

BlockEden.xyz は、Namefi ベースのアプリケーションへ直接統合できる、拡張性の高いマルチチェーン API を通じて、これらすべてを提供します。

### **共有するビジョン**

BlockEden.xyz と Namefi はどちらも、分散型インターネットを実世界のユーザーにとってアクセスしやすく使いやすいものにすることに取り組んでいます。Namefi がドメイントークン化によって Web2 と Web3 をつなぐ一方で、BlockEden.xyz はその背後でシステムが高速かつ回復力を持ち、本番運用に耐える形で確実に動くよう支えます。

Namefi エコシステムの成長に合わせて、BlockEden.xyz は必要な開発者ツールとインフラを提供し続けます。
