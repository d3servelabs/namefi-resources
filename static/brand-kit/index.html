Logo Rules

Use these rules to keep Namefi consistent, recognizable, and clear across every placement.

1

Use green first

The primary logotype should use #1CD17D on dark or light neutral fields when contrast holds.

2

Reserve mono

Use the white mono mark on dark photography, dense UI chrome, or partner lockups where green competes.

3

Keep clearspace

Use at least the height of the N counter block around the logotype. Do not crowd edges or badges.

4

Protect color intent

Avoid gradients, outlines, shadows, or arbitrary alternate hues for the main logotype. Use mono treatments only when the placement needs a single-color mark.

5

Spell Namefi

Use Namefi in running text. Do not write NameFi or NameFI. Use NAMEFI only where full capitalization is required.
