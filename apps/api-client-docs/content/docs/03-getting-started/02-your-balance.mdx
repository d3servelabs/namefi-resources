## Request NFSC Faucet (Test)

Use `client.user.requestNfscFaucet` to request test NFSC.
If you refer to this as `requestNFSC` internally, this is the client method to call.

<CodeTabs items={["Node.js SDK", "Node.js fetch", "cURL", "Python"]}>
<Tab value="Node.js SDK">

```ts
import { createNamefiClient } from '@namefi/api-client';

const client = createNamefiClient({
  authentication: {
    apiKey: process.env.NAMEFI_API_KEY!,
    type: 'API_KEY',
  },
  logger: true,
});

const faucet = await client.user.requestNfscFaucet({});

console.log('Status', faucet.status); // started
console.log('Workflow ID', faucet.workflowId);
console.log('Wallet', faucet.walletAddress);
console.log('Next eligible at', faucet.nextEligibleAt);
```

</Tab>
<Tab value="Node.js fetch">

```ts
const res = await fetch('https://api.namefi.io/v-next/user/faucet', {
  method: 'POST',
  headers: {
    'x-api-key': process.env.NAMEFI_API_KEY!,
    'content-type': 'application/json',
  },
  body: JSON.stringify({}),
});
const faucet = await res.json();

console.log('Status', faucet.status); // started
console.log('Workflow ID', faucet.workflowId);
console.log('Wallet', faucet.walletAddress);
console.log('Next eligible at', faucet.nextEligibleAt);
```

</Tab>
<Tab value="cURL">

```bash
curl -X POST "https://api.namefi.io/v-next/user/faucet" \
  -H "x-api-key: $NAMEFI_API_KEY" \
  -H "content-type: application/json" \
  -d '{}'
```

</Tab>
<Tab value="Python">

```python
import os
import requests

faucet = requests.post(
    "https://api.namefi.io/v-next/user/faucet",
    headers={"x-api-key": os.environ["NAMEFI_API_KEY"]},
    json={},
).json()

print("Status", faucet["status"])  # started
print("Workflow ID", faucet["workflowId"])
print("Wallet", faucet["walletAddress"])
print("Next eligible at", faucet["nextEligibleAt"])
```

## Check Your Balance

Check balances with the API:

<CodeTabs items={["Node.js SDK", "Node.js fetch", "cURL", "Python"]}>
<Tab value="Node.js SDK">

```ts
import { createNamefiClient } from '@namefi/api-client';

const client = createNamefiClient({
  authentication: {
    apiKey: process.env.NAMEFI_API_KEY!,
    type: 'API_KEY',
  },
  logger: true,
});

const balance = await client.balance.getBalance();

console.log('Total USD', balance.totalBalanceInUsd);
console.log('Total USD cents', balance.totalBalanceInUsdCents);

for (const entry of balance.balances) {
  console.log(
    `${entry.chainName} (${entry.chainId}) - ${entry.walletAddress}: $${entry.balanceInUsd}`
  );
}
```

</Tab>
<Tab value="Node.js fetch">

```ts
const res = await fetch('https://api.namefi.io/v-next/balance', {
  headers: { 'x-api-key': process.env.NAMEFI_API_KEY! },
});
const balance = await res.json();

console.log('Total USD', balance.totalBalanceInUsd);
console.log('Total USD cents', balance.totalBalanceInUsdCents);

for (const entry of balance.balances) {
  console.log(
    `${entry.chainName} (${entry.chainId}) - ${entry.walletAddress}: $${entry.balanceInUsd}`
  );
}
```

</Tab>
<Tab value="cURL">

```bash
curl "https://api.namefi.io/v-next/balance" \
  -H "x-api-key: $NAMEFI_API_KEY"
```

</Tab>
<Tab value="Python">

```python
import os
import requests

balance = requests.get(
    "https://api.namefi.io/v-next/balance",
    headers={"x-api-key": os.environ["NAMEFI_API_KEY"]},
).json()

print("Total USD", balance["totalBalanceInUsd"])
print("Total USD cents", balance["totalBalanceInUsdCents"])

for entry in balance["balances"]:
    print(
        f"{entry['chainName']} ({entry['chainId']}) - "
        f"{entry['walletAddress']}: ${entry['balanceInUsd']}"
    )
```
