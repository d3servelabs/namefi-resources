description: How to set up and install the Namefi client
---

The Namefi client is still in alpha. APIs and behavior may change without notice.

[![npm version](https://img.shields.io/npm/v/@namefi/api-client.svg)](https://www.npmjs.com/package/@namefi/api-client)

## Install

```bash
npm install @namefi/api-client
```

## Import

```ts
import { createNamefiClient } from '@namefi/api-client';
```
