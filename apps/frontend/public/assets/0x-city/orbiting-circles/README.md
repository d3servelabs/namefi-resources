# Orbiting Circles

This folder contains static visual assets for the 0x City experience. The files are
served directly from `public`, so consumers should reference the stable URL path rather
than importing them as modules.

## File Relationships

- Files are served as static assets and may be referenced by stable URL paths.
- Prefer additive filename changes when assets are cached or linked from content.

## Structure

```text
apps/frontend/public/assets/0x-city/orbiting-circles/

```

Update this README when assets are reorganized, public URL conventions change, or a new
