# Kumar

This folder contains static CV-style media assets for the Kumar PBNS/persona experience.
Components reference these files by public URL, so keep filenames stable when updating
the visual set.

## File Relationships

- Files are served as static assets and may be referenced by stable URL paths.
- Prefer additive filename changes when assets are cached or linked from content.
