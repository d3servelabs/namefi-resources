# `/gallery` screenshots

Reference captures of the public NFT Gallery page at
`apps/frontend/src/app/gallery/page.tsx`. Generated against the PR-4241
preview deployment using agent-browser at 1440×900 (desktop) and 390×844
(mobile) viewports.

These are committed for two reasons:

1. Inline visual evidence in the originating PR (#4241).
2. Durable visual reference for future work on `/gallery` — useful when
   evaluating layout regressions or before/after comparisons.

Regenerate by re-running the capture against the current preview URL
when the page meaningfully changes (new sort/filter UI, layout swap,
etc.). Stale screenshots are worse than no screenshots — delete them
rather than letting them drift.
