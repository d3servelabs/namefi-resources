# English Frontend Messages

This folder contains the source English copy for frontend message namespaces.
Other locale folders mirror these files and key structures.

```text
en/
  common.json   # shared application UI strings
  nfsc.json     # Namefi Service Credit copy
  *.json        # feature or page message namespaces
```

Add new user-facing strings here first, then update every locale and the
matching metadata in `../_meta` when translator context is useful.
