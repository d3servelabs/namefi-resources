# Src Components

This folder contains React components for parked-domain pages. Components should stay
lightweight, SEO-friendly, and safe for public domain traffic.

## File Relationships

- Components in this folder are peers for the same feature or UI layer.
- Stories, tests, and local helpers should stay near the component they explain unless shared broadly.
