# Providers

This folder contains React components for the Resources app. Components here render
articles, navigation, providers, and content affordances for the localized resources
site.

## File Relationships

- Components in this folder are peers for the same feature or UI layer.
- Stories, tests, and local helpers should stay near the component they explain unless shared broadly.
