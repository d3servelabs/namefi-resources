# Progress Timeline

This folder contains Namefi progress-timeline UI pieces. Components here work together
to render multi-step status and progress experiences.

## File Relationships

- Components in this folder are peers for the same feature or UI layer.
- Stories, tests, and local helpers should stay near the component they explain unless shared broadly.
